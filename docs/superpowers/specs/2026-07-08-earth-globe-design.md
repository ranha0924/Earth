# 3D 지구본 사회공부 사이트 — 설계 문서

- **작성일**: 2026-07-08
- **대상**: 고1 통합사회 1학기 3~4단원(자연환경과 인간생활 / 문화와 다양성) 복습
- **한 줄 요약**: 3D 지구본에서 실제 기후대를 색으로 보고, 나라를 클릭하면 기후·환경문제·문화가 카드로 뜨며, 퀴즈로 스스로 확인하는 반응형 웹 학습 사이트.

---

## 1. 목적과 배경

고1 동생의 통합사회 3~4단원 암기를 돕는 학습용 웹사이트다. 이 단원의 핵심 개념(기후, 환경문제, 국제협약, 문화)은 모두 "지구 위 위치"와 밀접하게 붙어 있다. 그래서 교과서의 평면 기후 지도를 **살아있는 3D 지구본**으로 만들어, 위치와 개념을 함께 기억하게 하는 것이 핵심 가치다.

**설계의 핵심 결정**: 나라별 팝업에 모든 정보를 몰아넣지 않고, **레이어(모드) 전환** 방식을 쓴다. 정보를 "나라 단위"가 아니라 각 주제의 자연스러운 단위(기후=기후대 면, 환경=지역 현상, 축제=지점 핀)로 표현해, 교과서가 가르치는 패턴이 눈에 보이게 한다.

## 2. 학습 콘텐츠 범위

- **기후**: 열대(열대우림·열대몬순·사바나), 건조(스텝·사막), 고산, 온대(온난습윤·서안해양성·지중해성·온대겨울건조), 냉대(냉대습윤·냉대겨울건조), 한대(툰드라·빙설)
- **환경문제**: 산성비, 오존층 파괴, 열대림 파괴, 사막화, 미세먼지, 해양오염, 미세플라스틱
- **국제협약**: 람사르 협약, 몬트리올 의정서, 바젤 협약, 파리 협정, 런던 협약, 교토 의정서, 기후변화협약, 생물다양성 협약, 사막화 방지 협약
- **문화**: 가옥·의복·음식·종교, 축제 6종(스웨덴 하지, 인도 홀리, 브라질 리우 카니발, 에스파냐 부뇰 토마토, 타이 송끄란, 페루 쿠스코 태양제)

## 3. 아키텍처

**서버 없는 정적 웹앱.** 모든 학습 데이터가 미리 준비된 정적 파일이라 백엔드가 필요 없다. 무료 정적 호스팅으로 배포하며 관리가 단순하다.

**기술 구성**
- 빌드: Vite / 언어: React + TypeScript
- 3D: globe.gl (three-globe / Three.js 기반). 선택 이유 — 텍스처 지구본, 국경 폴리곤 클릭, 핀, 지역 하이라이트가 라이브러리에 기본 제공되어 목표 기능을 가장 빠르고 안정적으로 구현 가능
- 상태관리: Zustand — `mode`, `selectedCountry`, `quizState`만 관리
- 데이터: 정적 JSON + 쾨펜 기후 텍스처 이미지 + 국경 GeoJSON
- 스타일: 반응형 CSS (PC·모바일 모두 지원)

**컴포넌트 (역할 하나씩)**

| 컴포넌트 | 역할 | 입력 | 출력 |
|---|---|---|---|
| `GlobeView` | 3D 지구본 렌더. 모드에 따라 표면 텍스처/핀/하이라이트 전환 | `mode`, 데이터 | 나라/핀/문제 클릭 이벤트 |
| `ModeSwitcher` | 기후/환경/문화 + 퀴즈 전환 | 현재 `mode` | `mode` 변경 |
| `Legend` | 현재 모드 범례(기후 색표 / 환경 아이콘). 기후 범례는 클릭 필터 기능 | `mode` | 범례 항목 선택 → 강조 필터 |
| `InfoCard` | 선택한 나라 요약 카드 | `selectedCountry` | 상세 열기 |
| `DetailPanel` | 환경문제 → 원인·영향·관련 협약 상세 | 선택 문제 | — |
| `QuizPanel` | 퀴즈 모드 UI(문제·채점·해설·점수) | `quizState`, 데이터 | 답 판정 |

## 5. 모드 상세

### 5.1 기후 모드
- 지구본 표면 = 공개 쾨펜-가이거 기후 지도를 교과서 대분류 색으로 재채색한 텍스처. 대분류는 열대·건조·온대·냉대·한대 5개 + 고산(별도 색), 하위 기후는 같은 계열의 명암/무늬로 구분. 범례에도 고산을 별도 항목으로 표시.
- 색칠 단위는 **실제 기후대 경계선**(나라 단위 단색 아님). 한 나라 안에도 여러 기후가 보여 위도별 배열·대륙 서안 vs 동안 패턴이 정확히 드러남.
- 범례 항목 **클릭 시 해당 기후대만 강조**(나머지 흐리게) → "지중해성 기후 분포"를 한눈에.
- 나라 클릭 → 카드에 그 나라 기후 표시.

### 5.2 환경문제 모드
- 나라가 아니라 "지역 현상"이므로 지구본을 중립색으로 낮추고 지역 하이라이트 + 아이콘 표시.
- 지역 매핑: 산성비=유럽·동아시아·북미 공업지역 / 오존층 파괴=남극 상공 / 열대림 파괴=아마존·콩고·보르네오 / 사막화=사헬·중앙아시아 / 미세먼지=동아시아 / 해양오염=주요 연안 / 미세플라스틱=태평양 쓰레기 지대.
- 문제 클릭 → 상세 패널에 원인·영향 + **관련 국제협약**.

**환경 주제 ↔ 국제협약 매핑**

| 환경 주제 | 국제협약 |
|---|---|
| 오존층 파괴 | 몬트리올 의정서 |
| 지구온난화·기후변화 | 기후변화협약 · 교토의정서 · 파리협정 |
| 사막화 | 사막화 방지 협약 |
| 습지 보전 | 람사르 협약 |
| 유해폐기물 국경 이동 | 바젤 협약 |
| 해양 투기 | 런던 협약 |
| 생물다양성(열대림 등) | 생물다양성 협약 |

### 5.3 문화 모드
- **축제 핀 6개**: 스웨덴 하지 · 인도 홀리 · 브라질 리우 카니발 · 에스파냐 부뇰 토마토 · 타이 송끄란 · 페루 쿠스코 태양제. 핀 클릭 시 사진+설명 카드.
- **전통 가옥** 핀/지역: 이글루 · 게르 · 고상가옥 · 흙집 · 통나무집.
- **종교 분포**: 권역 오버레이(문화 모드 안 하위 토글).
- 의복·음식: 나라/권역 카드에 표시.
- **기후 연결**: 가옥·의복·음식은 기후의 결과임을 카드에서 명시(고상가옥=열대 습기, 게르=건조 유목, 이글루=한대) → 3~4단원을 하나로 꿰는 학습 포인트.

## 6. 나라 카드 내용

```
🇰🇷 대한민국 (South Korea)
🌡️ 기후   온대 · 온난습윤  "여름 덥고 습, 겨울 건조"
♻️ 환경   미세먼지, 해양오염
🎭 문화   한옥 · 온돌 · 김치
🔗 연결   온돌은 겨울 추위(온대) 때문에 발달
```
- **대표국**은 기후+환경+문화 3칸을 채우고, **일반 국가**는 기후만 표시하며 나머지 칸은 조용히 비움(억지로 채우지 않음).
- 항상 **기후–문화 연결 한 줄**을 넣어 단원을 꿰어 줌.

## 7. 퀴즈 모드

지구본이 그대로 문제판이 된다. 문제 유형 5가지:
1. **색→기후**: 지구본에 한 기후대만 칠하고 "이 기후는?" 4지선다
2. **위치→기후**: 특정 나라를 강조하고 기후 맞히기
3. **축제 찾기**: "리우 카니발은 어느 나라?" → 지구본에서 나라 클릭
4. **환경↔협약 짝짓기**: "오존층 파괴 대응 협약은?"
5. **가옥↔기후 연결**: "게르는 어떤 기후?"

즉시 채점 + 오답 시 짧은 해설 + 세션 점수 표시. 문제는 기존 데이터에서 자동 생성(콘텐츠 중복 제작 없음).

## 8. 데이터

**출처 (전부 무료/공개)**
- 기후 텍스처: 공개 쾨펜-가이거 기후 지도 → 교과서 5분류 색으로 재채색
- 국경: Natural Earth(퍼블릭 도메인) GeoJSON
- 환경·협약·문화 텍스트: 통합사회 수준으로 직접 조사·작성(한국어), 교과서 개념 기준 정확도 확인
- 축제·가옥 이미지: 저작권 안전한 CC/퍼블릭 이미지 또는 일러스트/이모지로 대체

**파일 구조**
```
public/textures/koppen.png        재채색 기후 지도
src/data/countries.geojson        국경
src/data/climate.json             나라→기후
src/data/environment.json         문제·지역·연결 협약
src/data/treaties.json            협약 9개
src/data/culture.json             축제·가옥·종교·의식주
src/data/quiz.json                문제 템플릿
```

**데이터 흐름**
- 앱 상태 `{ mode, selectedCountry, quizState }`를 Zustand가 보관.
- `ModeSwitcher`가 `mode` 변경 → `GlobeView`가 해당 레이어(기후=텍스처 / 환경·문화=핀·하이라이트)로 다시 그림.
- 나라 클릭 → climate/environment/culture 데이터를 조회해 `InfoCard` 렌더.
- 퀴즈 모드 → `QuizPanel`이 문제를 내고 지구본이 답안 표면이 됨.

## 9. 오류 처리 / 예외

- 데이터·텍스처 로딩 실패 → 친절한 재시도 안내 메시지.
- 바다/빈 곳 클릭 → 카드 닫힘.
- 대표국이 아닌 나라 → 기후만 표시, 나머지 칸은 조용히 비움.
- 모바일 성능 → 저해상 텍스처·핀 개수 제한.
- 색맹 배려 → 색에 텍스트 라벨 병기.

## 10. 테스트

- **단위(Vitest)**: 데이터 조회, 퀴즈 채점, 기후 매핑 등 순수 함수.
- **컴포넌트(React Testing Library)**: 모드 전환 시 올바른 레이어/카드 표시, 카드 필드 정확성, 퀴즈 채점.
- **E2E(Playwright)**: 지구본 로딩 → 나라 클릭 → 카드 표시, 모드 전환을 PC·모바일 뷰포트 양쪽에서 검증.

## 11. 개발 단계 (MVP부터 일찍 사용 가능)

1. **1단계 (MVP)**: 지구본 + 기후 모드(텍스처 + 범례 클릭 필터) + 나라 클릭 카드(기후). 이것만으로 바로 공부에 사용 가능.
2. **2단계**: 환경 모드 + 협약 연결.
3. **3단계**: 문화 모드(축제 핀·가옥·종교).
4. **4단계**: 퀴즈 모드.
5. **5단계**: 배포 + 마무리.

반응형·모바일 대응은 1단계부터 기본 포함. 배포는 1단계부터 가능.
