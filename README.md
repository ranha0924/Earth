# 세계 탐구 도감

고1 통합사회 1학기 3~4단원(자연환경과 인간 생활 / 문화와 다양성) 학습용 **3D 지구본** 웹사이트.
지구본을 돌려 나라를 누르며 기후·환경문제·국제협약·문화를 공부하고, 퀴즈로 확인할 수 있습니다.

## 모드

- **🌍 기후** — 쾨펜 기후 텍스처 지구본. 6개 기후 대분류 범례로 필터, 나라 클릭 시 기후(대분류·소분류) 정보 카드. (175개국)
- **⚠️ 환경 — 환경문제** — 산성비·오존층 파괴·열대림 파괴·사막화·미세먼지·해양 오염·미세 플라스틱 7가지. 문제 선택 시 발생·피해 지역이 지구본에 표시되고 원인→현상→영향→대책과 관련 협약을 정리.
- **🤝 환경 — 국제협약** — 람사르·런던·몬트리올·바젤·기후변화협약·생물다양성·사막화방지·교토·파리 9개 협약 연표(기후변화협약→교토→파리 계보 강조) + 협약↔키워드 매칭 게임.
- **🎭 문화** — 종교 분포 색칠(8분류)과 세계 6대 축제 핀(하지·홀리·리우 카니발·라 토마티나·송끄란·태양제). 나라 클릭 시 전통 가옥·의복·음식·종교를 기후와 연결해 설명.
- **✏️ 퀴즈** — 기후·환경문제·협약·문화 4지선다 + 지도에서 나라 찾기. 즉시 채점·해설·오답 노트, 점수는 브라우저(localStorage)에 저장.

## 개발

```bash
npm install
npm run dev      # 개발 서버
npm test         # 단위 테스트 (vitest)
npm run e2e      # E2E 테스트 (playwright)
npm run build    # 프로덕션 빌드 → dist/
```
